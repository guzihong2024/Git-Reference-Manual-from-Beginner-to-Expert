🚀 Git - 从入门到精通 (0 to Pro) 参考手册



这是一个基于 supersimple.dev 视频教程创建的 Git 命令和工作流参考手册。



本手册旨在提供一个清晰、结构化的速查卡片，帮助开发者快速掌握 Git 和 GitHub 的核心操作。


💡 文件内容



本仓库包含一个经过优化的 HTML 文件，旨在提供最佳的打印效果：



| 文件名 | 描述 |



| `git-reference-complete.html` | 包含了 Git 基础命令、工作区、分支、合并冲突以及 GitHub 工作流的完整指南。所有说明文字均为简体中文，而代码和命令保持英文原版。 |



✨ 特点


高还原度： HTML/CSS 结构最大程度模拟了原始 PDF 的卡片式布局和视觉流。

双语对照： 关键 Git 术语采用英文原文 + 简体中文翻译。

一页打印： 专为浏览器打印到 PDF 优化，便于查阅和学习。



🖨️ 如何生成 PDF



1\.  用任何现代浏览器（Chrome、Edge 等）打开 `git-reference-complete.html` 文件。

2\.  按下 `Ctrl + P` (或 `Cmd + P`) 打开打印对话框。

3\.  将目标打印机选择为"另存为 PDF" (Save as PDF)。

4\.  在打印设置中，建议：

纸张大小：A4 或 Letter

布局： 纵向 (Portrait)

边距：默认或无
